Sublime Text 手册

空白字符

  • Cmd+, 进入 Preference - Settings - User
  • 增加下列设置：
"tab_size": 4,
"translate_tabs_to_spaces": true,
"draw_white_space": all
  • Cmd+s 保存，Cmd+w 退出

    Vim 模式

  • 进入 Preference -Settings - Default
  • 找到：ignored_packages
  • 删除其中的 “Vintage”
  • Cmd+s 保存，Cmd+w 退出
